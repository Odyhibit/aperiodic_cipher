Algorithm Information

Word Length Aperiodic: Changes alphabet with each word instead of each letter. Uses a sequence of alphabets in rotation, with a new alphabet starting at each word.

Numerically Keyed Aperiodic: Uses a numerical key to determine which alphabet to use for each letter in the message.

Interruptor Letter Aperiodic: Uses a special letter that signals when to change to the next alphabet in the sequence.
